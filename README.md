# KnitStichGrid

Yup something like this probably exists, but I wanted to make it myself for fun. I am hoping to make simple app that can be used to create (flat) knit patterns regardless of gauge.

Overall idea:
- A grid that has an adjustable stitch and row gauge
- Option to overlay an image to trace
- Verify finished size based on the gauge inputted
- Support mixed cell sizes for rows of crochet
